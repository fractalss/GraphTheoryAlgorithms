### How to compile and run the program

1.Make sure you are in the directory of  program_files

2.  On your command terminal, type 
make

This will compile all the *.cpp files and generate the respective object codes

2. On your command terminal, type
./hw2

3. This will run the simulation first for 20% Graph Density and then press Enter to run it for 40% Graph Density

4. Send me an email if you come across difficulties.
